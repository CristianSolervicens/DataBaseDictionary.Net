# DataBaseDictionary.Net

## Author: _Cristian Solervicéns C._
## Platform: MS SQL Server

## Language: .NET Core 8 with Razor

## Description:
Simple program to create a Database Dictionary based on SQL Server Extended Properties.
It generates and HTML documentation file with variosus options where you can include or omit
the various supported database objects: Tables, Views, Stored Procedure, Views (scalar and tabular)

## Note:
It works together with SQLCrypt (you can download it from my Github account)
SQLCrypt is a simple and ligthweight, jet powerful SQL Server Client which allows you, among other things,
to edit SQL Server Extended Properties through a very simple interface.
Among their features you can save your frecuent queries in a special folder and execute them from a list
with dynamic parameters.
